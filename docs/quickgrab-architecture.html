Security
🔐
Auth Middleware
  • Verify JWT token on every request
  • Decode user ID and role
  • Reject expired or invalid tokens
🛡️
Role Middleware
  • Check role = "admin" or "customer"
  • Block wrong role from wrong route
  • Return 403 if unauthorized
🌐
CORS Middleware
  • Allow browser to call API
  • Whitelist frontend origin
  • Handle preflight requests
